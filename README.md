# Credit Card Approval Prediction

## Project Overview
Credit Card Approval Prediction is a Machine Learning project that predicts whether a credit card application will be approved or rejected based on applicant information. The system uses Machine Learning classification algorithms and a Flask web application to provide predictions.

## Objective
The objective of this project is to automate the credit card approval process using Machine Learning techniques and assist in faster and more accurate decision-making.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- CSS
- Jupyter Notebook / PyCharm

## Machine Learning Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost / Gradient Boosting

## Dataset
The project uses a credit card approval dataset containing applicant details such as gender, income, education, family status, housing type, employment details, and other financial attributes.

## Project Workflow
1. Dataset Collection
2. Data Analysis
3. Data Preprocessing
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Flask Web Application Development
8. Model Deployment

## Project Structure
The project contains app.py for the Flask application, credit_card_model.pkl for the trained machine learning model, credit_card.csv for the dataset, requirements.txt for required libraries, README.md for project documentation, templates folder containing index.html for the web page, static folder containing style.css for designing the web interface, and Project_Documentation.pdf containing detailed project information.

## How to Run the Project

Step 1: Install Required Libraries

pip install -r requirements.txt

Step 2: Run Flask Application

python app.py

Step 3: Open Browser

http://127.0.0.1:5000/

## Features
- Predicts credit card approval status
- User-friendly web interface
- Machine Learning based prediction
- Real-time approval prediction
- Flask-based deployment

## Output
The application predicts whether the applicant is eligible for credit card approval based on the entered details.

## Conclusion
This project demonstrates how Machine Learning can be applied in the banking and finance domain to automate credit card approval decisions and improve efficiency through data-driven predictions.
